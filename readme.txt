/*


NXCMS - 免费的多语言外贸企业网站系统 
--------------------------------------------


只有在 PHP 5.4+, 你可以使用[] 作为语法.





整体上设计没有太大问题，数据表结构设计类型、长度、字段提取等都比较可理，但没有支持考虑多语言支持，
需要稍加完善。现在我整理所有需要的设计的表信息，根据下面给定的表信息重新设计完善以支持多语言。


项目简述：
------------------------------------------------------------------------------------------------
NXCMS一个外贸网站系统，它支持多语言（默认支持：中英文）的企业网站，网站上的每个信息都有对应的语言版本。
如：“关于我们”会有对应中文、英文、日语等多种语言的介绍；

表信息汇总：
-----------------------------------------------------------------------------------------------------
1、系统设置信息，

2、系统标签：同一个东西表示地址，如：地址、address、其它语言的表示

3、系统语言信息：中文、英文、日语，阿拉伯等...

4、联系我们信息：公司网站名称，地址，IM工具，邮件地址，联系电话等

5、新闻分类
6、新闻信息

7、产品分类
8、产品信息

9、设备分类
10、设备信息

11、案例分类
12、案例信息

13、用户信息，用于登录后台管理系统
14、招聘信息
15、单页页信息，使用示例-如：关于我们，公司简介，营销网络

根据浏览器自动转换语言,




URL 规定：
-------------------------------------------------------------------

	index.php?m=news&a=index&id=id;

	news
	news/page-<id>.html

	news/details-<id>.html
	
	news/category-<id>.html?page=1
	news/category-<id>.html



	index


































app = 默认值是: home
act = 默认值是: index

如果 app 的值不提供，那么默认给它赋值为 home
如果 act 的值不提供，那么默认给它赋值为 index


/index.php?app=home&act=user => /home/user/view-720.html
/index.php?app=admin&act=user-list
/index.php?app=admin&act=user-edit
/index.php?admin&act=user-remove


*/
